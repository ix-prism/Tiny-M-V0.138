# Tiny-M + Voron0.2 Mod
a mod for Voron0/Tiny-M 3D printer/一个基于Voron0/Tiny-M的3D打印机mod

# 与Tiny-M的区别

* Voron 0.2风格化

* MiniFrontBurner打印头，支持E3dV6或Bambu热端

* X轴导轨替换为mgn9h，安装在侧面

* 挤出机为Sherpa Mini

* 屏幕替换为V0Display
