# Tiny-M_V0.138_Voron0_Mod
a mod for Voron0/Tiny-M 3D printer/一个基于Voron0/Tiny-M的3D打印机mod

·MiniFrontBurner，支持E3dV6或Bambu热端

·X轴导轨替换为mgn9h

·挤出机替换为Sherpa Mini

·屏幕替换为V0Display
