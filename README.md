# ix_v0.138_Tiny-M_Voron0_Mod
a mod for Voron0/Tiny-M 3D printer/一个基于Voron0/Tiny-M的3D打印机mod

RC2更新：

·喷头替换为MiniFrontBurner，热端使用龙或E3d V6打孔攻丝

·X轴导轨替换为mgn9h

·挤出机替换为Sherpa Mini

·可选双推力轴承解耦

·屏幕替换为V0Display

·主板替换为SKR Mini E3
